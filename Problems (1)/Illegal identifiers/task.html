Illegal identifiers Open on Hyperskill

Suppose, you want to create a new programming language that supports variables.

There is a set of rules for identifiers (names) of variables:
  • it can include lower and upper letters, digits and the underscore character _;
  • it cannot start with a digit;
  • if an identifier starts with the underscore, the second character must be a letter or digit, but not the underscore;
Note, a single _ is not a valid identifier.

Using the provided template, write a program that read n identifiers and then outputs all illegal in the same order as they appear in the input data. We believe that you will use regexes to solve the problem.

Sample Input:
6
ident
i
__
1a
b33
_n1
Sample Output:
__
1a



Memory limit: 256 Mb
Time limit: 5s
