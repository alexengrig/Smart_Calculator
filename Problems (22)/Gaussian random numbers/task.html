Gaussian random numbers Open on Hyperskill

Find the first seed that is greater or equal to K where each of N Gaussian numbers is lesser than or equal M.

The input contains numbers K, N, M in a single line.

You should output the seed.

[HINT]You have to check all n iterations of Gaussian generated numbers to be less than M. If that is true, then escape loop and print the seed. If any of the generated numbers is greater than or equal to M, then you have to test out the next seed and go to the loop all over again.[/HINT]

Don't get too upset if you still don't understand this problem. Skip it or google for more information.


Sample Input:
0 5 0
Sample Output:
38

Sample Input:
0 5 -1.5
Sample Output:
498666

Sample Input:
10000 1 1.9
Sample Output:
10000


Memory limit: 256 Mb
Time limit: 5s
